Welcome to **Rat Detective Online**, a fast-paced, noir-inspired multiplayer detective game where you play as a stylish rat in a glowing purple city.

## Features
- **Noir Aesthetics**: A beautiful low-poly city with emissive materials and dramatic lighting.
- **Dynamic Movement**: Smooth character control with jumping and sprinting.
- **Cheese Gun Combat**: A unique projectile system with glowing cheese balls.
- **Ragdoll Physics**: Dramatic death sequences with physics-based flight.
- **Unique Rats**: Procedurally generated rat combinations (375+ possibilities).

## Getting Started
1. Clone the repo.
2. Run `npm install`.
3. Start the dev server with `npm run dev`.

## Tech Stack
- **Three.js**: Visual engine for 3D rendering.
- **Cannon-es**: Physics engine for rat behaviors and projectiles.
- **TypeScript**: Robust logic and type safety.
- **Vite**: Ultra-fast build tool.
